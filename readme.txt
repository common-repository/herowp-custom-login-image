Change wordpress login logo with your own.

== Description ==

If you want to change your default wordpress login logo with your own this is the right plugin for you. You can also add:

* Custom width
* Custom height


== Installation ==

Pretty simple. Just follow this steps

1. Upload folder `herowp-custom-login-image` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On the left menu in Wordrpress, you will see a menu called Admin Logo.
4. Upload a logo and click save.
